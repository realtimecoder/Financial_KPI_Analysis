# Financial KPI Analysis Dashboard

## Project Overview
This project analyzes financial performance using Power BI.

## KPIs Used
- Revenue
- Revenue Growth
- Gross Profit
- Net Income
- EBITDA
- Total Assets
- Total Liabilities

## Tools Used
- Power BI
- CSV Dataset
- GitHub

## Dashboard Features
- KPI Cards
- Revenue Trend Analysis
- Revenue Growth Analysis
- Profitability Analysis
- Assets vs Liabilities Comparison

## Key Findings
- Revenue fluctuated across years.
- Revenue growth showed varying trends.
- Assets exceeded liabilities, indicating stability.
- EBITDA reflected operational efficiency.

## Conclusion
The dashboard provides insights into financial performance and supports strategic business decision-making.
